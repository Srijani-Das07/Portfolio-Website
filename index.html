Implemented and compared CNN, Vision Transformer (ViT), and hybrid ViT+CNN architectures for multiclass classification on the PlantVillage dataset, achieving ~99% accuracy. Applied Grad-CAM visualizations to interpret model predictions and enhance explainability.

Voice Notes Summarizer

September 2025

A local Python application that converts audio files into complete transcriptions and generates summaries in multiple lengths (short, medium, detailed). The system uses Whisper for speech-to-text processing and BART for text summarization, with all computation performed on the user's device to ensure privacy. It supports common audio formats (MP3, WAV, M4A, FLAC, OGG) and enables downloads of both transcript and summary as text files, running entirely locally without APIs or external services.

Python Streamlit CSS OpenAI Whisper HuggingFace BART PyTorch
